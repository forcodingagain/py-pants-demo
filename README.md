# py-pants-demo

## common commands

* get build files

  ```bash
  pants tails ::
  ```

* fix

  ```bash
  pants fix ::
  ```

* fmt

  ```bash
  pants fmt ::
  ```

* type check

  ```bash
  pants check ::
  ```

* lint

  ```bash
  pants lint ::
  ```

* test

  ```bash
  pants test ::
  ```

* run

  ```bash
  pants run xxxx
  ```
